# 💰 Совместный учет расходов

Веб-приложение для совместного учета расходов, накоплений и баланса. Создано для пары (Андрей & Настя).

## ✨ Возможности

- ✅ Добавление расходов с указанием категории, суммы и фото чека
- ✅ Фильтрация расходов по категории, месяцу, году
- ✅ Статистика расходов по каждому человеку
- ✅ Управление накоплениями (добавление, удаление, история)
- ✅ Расчет текущего баланса (накопления минус расходы)
- ✅ Настройка цели накоплений
- ✅ Загрузка изображений чеков (до 5 МБ)

## 🛠 Технологии

- **Backend**: Node.js, Express 5, SQLite3, Multer, CORS
- **Frontend**: HTML, CSS, Vanilla JavaScript
- **Хранение**: SQLite (файл `expenses.db` создается автоматически), файловая система для изображений

## 📦 Установка и запуск

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/Waspish/expenses-app.git
cd expenses-app
```

### 2. Установите зависимости
```bash
npm install
```

### 3. Запустите сервер

**Через Node.js:**
```bash
node server.js
```

**Или через start.bat (Windows):**
```bash
start.bat
```

### 4. Откройте приложение в браузере
```
http://localhost:3000
```

## 🗄️ Как работает база данных и загрузки

- База данных (`expenses.db`) и папка для загрузок (`uploads/`) создаются автоматически при первом запуске.
- В `expenses.db` создаются три таблицы:
  - `expenses` — расходы
  - `categories` — категории (заполняются по умолчанию)
  - `savings` — накопления
- Загруженные изображения сохраняются в `uploads/` с уникальными именами.

## 📡 API эндпоинты

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/expenses` | Получить расходы (с фильтрацией по `category`, `month`, `year`) |
| POST | `/api/expenses` | Добавить расход (поля: `person`, `amount`, `description`, `category`, файл `image`) |
| DELETE | `/api/expenses/:id` | Удалить расход и связанное изображение |
| GET | `/api/balance` | Получить баланс (накопления - расходы) по каждому человеку |
| GET | `/api/categories` | Список доступных категорий |
| GET | `/api/statistics` | Статистика расходов по человеку |
| GET | `/api/available-years` | Список лет с расходами |
| POST | `/api/savings` | Добавить запись о накоплении |
| GET | `/api/savings` | История накоплений (пагинация через `limit`, `offset`) |
| GET | `/api/savings/total` | Общая сумма накоплений по каждому человеку |
| DELETE | `/api/savings/:id` | Удалить запись о накоплении |

## ⚙️ Переменные окружения

Проект не использует файл `.env`. Порт сервера жестко задан в коде (`3000`). При необходимости измените `PORT` в `server.js`.
